## 🇬🇧 Welcome to my digital workspace

I am a Software Engineering student driven by pure curiosity. I build things simply because I want to understand how they work under the hood—ranging from low-level systems to full server infrastructures. I code out of passion, constant study, and the sheer fun of solving complex engineering puzzles.

### 🚀 What I'm Currently Working On
* **EFOR (In Progress):** A custom command-line file explorer written from scratch in C. I am developing it to deeply understand memory buffers, terminal rendering, and state-reconciliation patterns between user input and the operating system's raw file system.

### 💻 Projects & Experience Highlights
* **Compiler Construction:** Building a basic compiler from scratch, scaling it step-by-step into a comprehensive tool.
* **Systems Administration & Infrastructure:** Deployed and secured multi-platform enterprise services across Windows Server and Ubuntu Server (DNS, DHCP, FTP/FTPS, mobile profiles, and MFA/OTP authentication via mobile tokens).
* **IoT & Computational Logic:** Android + Arduino Bluetooth integrations with optimized data streams, alongside a Turing Machine Simulator in JavaFX.

## 🇪🇸 Bienvenido a mi espacio de trabajo

Soy un estudiante de Ingeniería de Software impulsado por la pura curiosidad. Construyo software simplemente porque quiero entender cómo funcionan las cosas por dentro, desde sistemas de bajo nivel hasta infraestructuras de servidores completas. Programo por pasión, por estudio constante y por el simple gusto de resolver rompecabezas de ingeniería complejos.

### 🚀 Lo que estoy desarrollando actualmente (En proceso)
* **EFOR (En desarrollo activo):** Un explorador de archivos en línea de comandos escrito desde cero en C. Lo estoy creando para comprender a fondo los búferes de memoria, el renderizado en terminal y los patrones de reconciliación de estado entre las órdenes del usuario y el sistema de archivos real.

### 💻 Proyectos y Experiencia Destacada
* **Construcción de Compiladores:** Desarrollo de un compilador básico desde cero, escalándolo paso a paso hacia una herramienta más completa.
* **Administración de Sistemas e Infraestructura:** Despliegue y configuración de servicios empresariales multiplataforma en Windows Server y Ubuntu Server (DNS, DHCP, FTP/FTPS, perfiles móviles y autenticación MFA/OTP validada mediante tokens en el celular).
* **IoT y Lógica Computacional:** Integraciones Android + Arduino vía Bluetooth con optimización de datos, junto con un Simulador de Máquina de Turing en JavaFX.
