Soy **estudiante de Ingeniería de Software** y me apasiona entender cómo funciona la tecnología desde sus cimientos. No solo busco escribir código, sino comprender la arquitectura detrás de él: desde la lógica de compiladores y máquinas de Turing, hasta bases de datos distribuidas e integraciones entre móviles y hardware.

Actualmente estoy organizando este espacio para mostrar mis mejores proyectos, pasando de tener repositorios de prueba privados a construir un portafolio profesional sólido.
